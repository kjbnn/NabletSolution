
Описание функционала

1. Скопируйте папку \NabletSolution с проектом в корень c: 
2. Перейдите в папку C:\NabletSolution\Debug, Вам будут доступны следующие 4 программы.
3. demuxer.exe извелает треки из mp4 файла по номеру трека (0-31) и помещает в файл .\Debug\dump.tmp.
4. acc_dec.exe конвертирует acc в PCM.
5. acc_en.exe конвертируsт PCM в acc.
6. muxer.exe упаковывает дорожки в mp4 файл.

На мой взгляд еще можно поиграться настройками SDK, но на это уже нет времени.
